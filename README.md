## 🌐 Веб‑страница проекта

Проект доступен онлайн через GitHub Pages:  
[Открыть сайт](https://allanazarovyuldosh-tech.github.io/task-manager)

### 📂 Структура
- `README.md` — описание проекта и инструкция по запуску.
- `index.html` — стартовая страница для GitHub Pages.
- `src/main.py` — минимальный код на Python.

### 🚀 Как работает сайт
Файл `index.html` лежит в корне репозитория и отображается как веб‑страница.  
Сейчас он показывает заголовок **Task Manager запущен!**, приветственный текст и кнопку «Добавить задачу».  
При нажатии на кнопку появляется сообщение: *Задача добавлена!*.
