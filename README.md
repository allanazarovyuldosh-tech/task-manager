# Task Manager

## 🎯 Цель проекта
Создать простой и удобный менеджер задач, который позволит:
- Добавлять новые задачи
- Отмечать выполненные
- Удалять ненужные
- Хранить список задач локально или в базе данных

## 📂 Структура
- `src/` — исходный код
- `docs/` — документация
- `assets/` — ресурсы (иконки, изображения)
